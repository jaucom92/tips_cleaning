1. Descripción del dataset.¶

El dataset a analizar consiste en una recoleccin de las "propinas" generadas por un restaurante. En este podemos encontrar todo un conjunto de informacin multivariante: la cantidad de propina dejada por un cliente, el gnero del cliente, si este fumaba o no, etc.

El dataset ha sido descargado del siguiente enlace (https://github.com/mwaskom/seaborn-data/blob/master/tips.csv) y ha sido levemente modificado para poder realizar un proceso de limpieza mnimo.

2. Integración y selección de los datos de interés a analizar.¶

Vamos a analizar la relacin que existe entre la cantidad de propina dejada por cada usuario (tip) y las siguientes variables: total_bill (importe total de la comida), sex (genero) y day (da de la semana); con los dos siguientes objetivos:

  • Exponer las diferencias (si existen) entre los grupos y la cantidad de propina.
  • Predecir la propina (target) dada las anteriores variables (features).

Para ello, vamos a recodificar la variable sex (0=Female, 1=Male), distinguir la variable day(entre 'fin de semana'=0 y 'entre semana'=1) y agrupar la variable total_bill. Como podemos observar la variable day solo tiene ocurrencias en los das jueves, viernes, sbado y domingo.

def recode_gender(gender):
    
    if gender == 'Female':
        return 0

    elif gender == 'Male':
        return 1

    else:
        return np.nan
    
    
def recode_day(day):
    
    if day in ['Sat', 'Sun']:
        return 0

    elif day in ['Thur', 'Fri']:
        return 1

    else:
        return np.nan   
    
    
def recode_bill(bill, lim=1000):
    
    b = np.round(bill)
    
    if b <= 10:
        return 0

    elif b <= 20:
        return 1
    
    elif b <= lim:
        return 2

    else:
        return np.nan  
    

tips['sex'] = tips.sex.apply(recode_gender)
tips['day'] = tips.day.apply(recode_day)

tips = tips[['tip', 'total_bill', 'sex', 'day']]

3. Limpieza de los datos.¶

A continuacin, vamos a interrogarnos sobre la presencia de valores perdidos o extremos. Como veremos en la siguiente tabla, el dataset contiene 244 filas y presenta algunos valores vacos en las dos primeras columnas. Adems, podemos observar que ambas son de tipo objeto (carcter) debido a la presencia del smbolo $.

Por lo tanto, vamos a eliminar aquellas filas con valores nulos as como corregir los valores numricos (podemos eliminarlas debido a la naturaleza del anlisis - son imputaciones propias).

Respecto a los valores extremos, podemos prever la existencia de estos en la variable tips y total_bill (podemos suponer que habr algunos clientes con cuentas elevadas o "tips elevados"); debido a la no normalidad de ambas variables - hecho observable en la seccin 4.2 - vamos a utilizar la distancia interquartil para identificar estos valores (observaciones superiores o inferiores a 3 * IQR); para, acto seguido, reemplazarlos por una medida de centralidad como la mediana debido a la distribucin de las variables.

def outliers_to_na(series):
    s = series.copy()
    sup, inf = np.percentile(s, 75) + 3 * iqr(s), np.percentile(s, 25) - 3 * iqr(s)
    s.loc[np.logical_or(s > sup, s < inf)] = s.loc[~np.logical_or(s > sup, s < inf)].median()
    return s

tips['tip'] = outliers_to_na(tips.tip)
tips['total_bill'] = outliers_to_na(tips.total_bill)
